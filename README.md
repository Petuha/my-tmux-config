# Установка

```bash
sudo pacman -S tmux
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
mkdir -p ~/.config/tmux
cd ~/.config/tmux
git clone https://github.com/Petuha/my-tmux-config .
```

При первом входе установить плагины: `<prefix> + I`

# Бинды

`<prefix>` - `CTRL + SPACE`

`ALT + SHIFT + Left/Right` - переключение окон

`<prefix> + n` - новое окно

`<prefix> + ;` - ввести промт
